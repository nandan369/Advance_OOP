Detailed Description

A class that provides encryption and decryption using the Vigenère cipher.

The Vigenere class allows users to encrypt and decrypt messages using a key-based Vigenère cipher. The cipher shifts each letter in the message forward or backward based on corresponding letters in a repeating key.
